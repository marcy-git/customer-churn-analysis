# 📉 Customer Churn Analysis — Telecom Dataset

## Project Overview

Analyzed customer churn behavior for a telecom company (Databel) using Excel, 
identifying key drivers behind a 26.8% churn rate across 6,687 customers.

## Business Problem

Customer churn is one of the most critical KPIs for subscription-based businesses. 
Understanding *why* customers leave — and *who* is most likely to churn — enables 
companies to take proactive retention actions and reduce revenue loss.

## Solution

Built an Excel dashboard that:
1. **Calculates core KPIs**: Total customers, churned customers, overall churn rate
2. **Segments churn by demographics**: Age group (Under 30, Senior, Other), Gender
3. **Analyzes contract patterns**: Month-to-Month vs One Year vs Two Year contracts
4. **Evaluates plan behavior**: International plan usage and data consumption groups
5. **Identifies churn reasons and categories**: Root cause breakdown

## Tools & Techniques

| Tool | Usage |
|---|---|
| **Microsoft Excel** | Data analysis, dashboard, pivot tables |
| **Power Query** | Data transformation and aggregation |
| **Pivot Tables** | Segmentation and KPI calculation |
| **IF / XLOOKUP formulas** | Data classification and logic |

## Dataset

- **6,687 customers** with 30 variables
- Source: DataCamp practice dataset (Databel Telecom)
- Fields: Demographics, contract type, payment method, usage, churn reason

## Key Findings

- Overall churn rate: **26.8%**
- Month-to-Month contracts show significantly higher churn than annual contracts
- Senior customers and customers without unlimited data plans churn at higher rates
- Top churn categories identified for targeted retention strategy
