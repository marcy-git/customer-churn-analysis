# 📉 Customer Churn Analysis — Telecom Dataset

## 🔍 Overview
Excel dashboard analyzing customer churn behavior for a telecom company,
identifying key drivers behind a **26.8% churn rate** across 6,687 customers.

## 🛠️ Tools
- Microsoft Excel · Power Query · Pivot Tables

## 📊 Dataset
- 6,687 customers · 30 variables
- Source: DataCamp — Databel Telecom (public dataset)

## 💡 Key Findings
- Overall churn rate: **26.8%**
- Month-to-Month contracts churn significantly more than annual contracts
- Senior customers show higher churn rates
- Top churn reasons identified for retention strategy
